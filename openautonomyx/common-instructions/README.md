# common-instructions

Shared instruction layer for OpenAutonomyX.

## Owns
- Boilerplates for execution.
- Review, docs, and audit prompts.
- Shared agent-role instructions.
- Canonical repo prompts for cross-platform shared work.

## Does Not Own
- Product implementation prompts for OpenDataWorld/AgentNxt.
- Non-code assets (shared-assets).
- Reusable code/config artifacts (shared-repos).
- Org-wide strategy/architecture/PRD content (org-docs).
