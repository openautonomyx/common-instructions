# OpenAutonomyX High-Level PRD

## Purpose
OpenAutonomyX is the shared control layer above OpenDataWorld and AgentNxt. It standardizes cross-repo instructions, assets, reusable repo artifacts, and organization documentation.

## In Scope
- Shared instructions and boilerplates.
- Shared non-code assets (templates/schemas/examples/policies/design).
- Shared code/config/deploy scaffolding for future reuse.
- Organization-level PRD, strategy, architecture, ADRs, and review logs.

## Out of Scope
- Product app implementation in OpenDataWorld or AgentNxt.
- Platform-specific prompt registries in product repos.

## Success Criteria
- Four repos with explicit, non-overlapping responsibilities.
- Discoverable README and index in shared operational repos.
- Minimal duplication and clear boundary guidance.
